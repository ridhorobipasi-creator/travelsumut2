Objective:
Bangun sebuah platform website wisata Indonesia yang scalable, modular, dan modern, dengan fitur lengkap untuk pengguna dan admin. Sistem harus fleksibel dalam pengaturan wilayah, kategori, dan konten.

1. Tech Stack (Wajib Digunakan)

Backend: Laravel (API-first, REST/GraphQL opsional)

Frontend:

Next.js (SSR untuk SEO halaman publik)

React.js (dashboard admin SPA)

Database: MySQL / PostgreSQL

State Management: Redux / Zustand

Styling: Tailwind CSS / ShadCN UI

Auth: JWT / Laravel Sanctum

Deployment-ready (Docker optional)

2. Konsep Utama Sistem

Platform wisata berbasis wilayah Indonesia (Provinsi → Kota → Destinasi)

Semua konten dapat dikelola dari admin (dynamic, bukan hardcoded)

User bisa:

Booking paket wisata

Rental mobil

Custom trip request

Admin bisa mengatur:

Harga

Jadwal

Kategori wilayah

Konten website
